# STM32F103 基本框架 
## 环境参数
* Linux   
* arm-none-eabi-gcc    
* stm32 使用官方启动文件   

## 使用方法    
* 将源文件放入usr文件夹，makefile中添加相应的文件即可。  
* Makefile已经详细说明一些参数的修改。

## 注意事项
* 如果文件夹命名出现空格,编译会提示找不到路径,因此工程文件名尽量避免特殊符号
